Was ist CDCC?

CDCC (Central Database for Covid Cases) ist eine Zentrale Datenbank, welche für die Verwaltung der Testergebnisse für COVID-19 verwendet wird. Durch eine Zentralisierung der Daten kann eine einfache Kommunikation zwischen den Teststellen, den Laboren und dem Gesundheitsministerium sicher gestellt werden. Außerdem werden Sie sofort benachrichtigt, wenn ihr Test ausgewertet wurde.
